2需求文档（PRD）

面向产品、设计与研发的正式需求说明，定义“做什么”与“做到什么程度”。

2.1 产品概述

背景：用户（在职、工作日仅约 30 分钟/天）希望系统性提升英语语法、消除中式英语，并覆盖日常沟通与旅游场景。痛点是传统背单词枯燥、缺反馈、难坚持。

目标：打造一套纯前端、零安装、手机电脑皆可随时练的英语学习工具集，以“游戏化 + 间隔重复 + 即时解析”提升效率与趣味性。

价值：① 多模块覆盖输入（手册）→ 记忆（大闯关/拼写）→ 测评（测试达人）→ 规划（计划/模块）的完整闭环；② 艾宾浩斯 SRS 让复习“恰到好处”，不浪费时间；③ 每题详细解析直击中式英语根源。

5关键交互详细规则（补全）

针对用户最关心的「点已掌握之后到底什么规则、什么频率」以及各模块核心交互，逐条给出可执行、可验证的精确规则（均来自源码核对）。

5.1 测试达人 · SRS 间隔重复与「已掌握」机制（重点）

状态模型：每道题在本地存两份状态——srsState[id]={box:0..6, due:绝对时间戳} 与 masteredSet（已掌握集合）。box 决定复习档位，due 决定“哪天才能再出现”。

5.1.1 升阶与复习频率表（连对次数递减）

当前阶段 box升阶所需连对次数达标后动作下次复习间隔（出现频率）
新题 box0（立即可练）3 次升 box12 天后才再出现（频率：最低→每 2 天一次）
box1（2 天档）2 次升 box24 天后
box2（4 天档）1 次升 box37 天后
box3（7 天档）1 次升 box415 天后
box4（15 天档）1 次升 box530 天后
box5（30 天档）1 次升 box660 天后
box6（60 天最高档）1 次标记「已掌握」due=∞，默认不再出现
“频率”含义：频率 = 该题再次出现的节奏。box0 时“每次练习都可能抽到”（最高频）；越往上升，间隔越长、出现越稀；box6 掌握后频率=0（永不出现，除非手动移出）。

5.1.2 答错降级规则

  • 答错 → box 降 2 阶（最低到 0），due=now（立即可练），连对计数清零。
  • 若该题原本已“已掌握”，答错会自动把它移出 masteredSet，重新进入活跃池从头练。

5.1.3 漏学顺延规则（举例）

due 是绝对时间戳。某题“2 天后该复习”但你那天没打开工具，之后任意一天打开，它立即变为“已到期·可练”帮你补上；一旦你答了它，下一档间隔（4 天）从当天重新算，而不是从原定那天算。即：漏学不丢进度，只把整个节奏往后平移。

用户原话举例复述：第 1 天学完 → 原定第 3 天复习但没学 → 第 N 天来学，立即把漏的学了；之后的“4 天”从今天（第 N 天）起算。✅ 代码已实现（s.due = Date.now() + interval*DAY）。

5.1.4 「已掌握」面板交互（用户重点问的）

  1. 入口：首页统计块“已掌握 N 题 · 点击管理”→ 点击打开管理弹窗（modal）。
  2. 列表：弹窗内逐条列出已掌握题（题干摘要 + 红色“移出”按钮）。
  3. 点“移出”按钮 → 规则：
    • 从 masteredSet 删除 → 该题恢复活跃；
    • SRS 重置：box=0, due=now → 从头重走整条曲线（新的一轮：需连对 3 次→2 天→…→60 天再掌握）；
    • 即频率回到最高：移出后它又会频繁出现，直到再次走完曲线被标掌握。
  4. 这就是用户说的“下一个曲线轮次还是要出现”——移出 ≠ 删除，而是开启新的一轮复习周期。

5.1.5 主动标记「已掌握」（题头/反馈区按钮）

  • 做题时题目区有“已掌握”按钮，未作答点或答完后点都会调用 manualMarkMastered()。
  • 效果：当前题立即加入 masteredSet，提示“✅ 已标记为掌握，以后不再出现”，首页计数 +1。
  • 与自动掌握等效：之后默认不再出现，除非到面板“移出”。

5.1.6 出题过滤规则

模式是否出现已掌握题说明
极速闯关 / 生存挑战 / 填空听写 / 辨析判断 / 综合大挑战排除默认不再出现已掌握题
收藏练习只看收藏与掌握状态无关，仅练收藏的题

5.1.7 「跳过该阶段」按钮（新增 v20260714-1040）

每道题头部新增蓝色 ⏭ 跳过该阶段 按钮（已掌握模式隐藏）。作用与「标掌握」严格区分：

  • 只升一阶：点击后当前题 box + 1，并推后到下一档间隔（从今天起算，支持漏学顺延）；下一阶段到期后仍会按曲线再次出现。
  • 不标记掌握：不加入 masteredSet，不会像「标掌握」那样永久消失（除非去「已掌握」面板移出重走）。
  • 不计入错题：未作答跳过记为「跳过本阶段」，不扣生存挑战生命值。
  • 边界（box6/60天档）：已是最高档时点此按钮 = 进入「已掌握」（下一阶段即掌握），仍可在面板移出重走。
  • 与出题过滤联动：因 due 推到未来，本题立即离开当前 due 池，本场不会立刻再现，直至下一阶段到期。

5.1.8 每题历史条显示文案（规则可视化）

  • 未掌握：📊 历史：✅答对 c ❌答错 w 正确率 x% + 频率提示：
    • box0：🔁 连对 st/3 次后延长间隔
    • 已排期：⏰ 已排期：X 分钟后复习（X 可为分钟/小时/天）
    • 已到期：⏰ 已到期 · 连对 st/N 次后延长间隔（box6 显示“后标为已掌握”）
  • 已掌握：📊 历史：✅ 已掌握（对 c / 错 w）· 默认不再出现，可在「已掌握」面板移出

5.1.9 云同步

SRS 状态与 masteredSet 通过 GitHub Gist 同步多端：box 取各端最大值，masteredSet 取并集，due 按本机时间重算，避免冲突丢失。
