🎯 现在完成时 have/has + done 完整解析 重点突破
一、核心公式
句型结构例句
肯定句 主语 + have/has + 过去分词 I have visited Milan.
否定句 have/has + not + 过去分词 I haven't been there.
疑问句 Have/Has + 主语 + 过去分词? Have you been to Hong Kong?
have vs has：三单主语（he/she/it/单数人名）→ has；其余（I/you/we/they/复数）→ have
两种形式：① have/has + 实义动词过去分词（普通动作）② have/has + been（be 动词的完成式）
二、核心本质：过去动作 → 留下现在结果/影响
不关心动作具体哪一天发生，只看重两点其一：
① 动作发生在过去，但对当下有影响
② 动作从过去持续到现在
时态例句侧重点
一般过去时 I went there in 2019. 带过去时间，只陈述当年去过，和现在无关
现在完成时 I've been there. 不带过去时间，重点：我现在拥有「去过那里」的经历
三、4大用法
用法1：曾经有过某种经历（最常用）
含义：人生到目前为止做过某事，常搭配 ever / never / before
拆解：I've been there.
I = 主语  |  have = 助动词（现完标志） |  been = be 的过去分词（be → was/were → been） |  there = 地点状语

直译：我到过那里 / 我去过那儿
现在影响：我现在熟悉这个地方，不用给我介绍；我有这段旅行经历
例句翻译
She's been to Australia.她去过澳大利亚。
Have you ever been to Hong Kong?你去过中国香港吗？
固定搭配：have been to + 地点 = 去过某地，现已回来
用法2：过去动作，结果留在当下（有直观影响）
例句翻译现在影响
I have lost my wallet.我钱包丢了。现在没钱，没法付款
We have eaten dinner.我们吃过晚饭了。现在不饿，不用点餐
用法3：动作从过去持续到现在（常搭配 for / since）
例句翻译时间起点
We have lived together for six years.我们同居六年了。2020年开始，持续到现在
I have worked in Shenzhen since March 2019.我2019年3月起就在深圳工作。至今还在
for vs since：for + 一段时间（for 3 years）/ since + 过去时间点（since 2019）
用法4：刚刚完成的动作（搭配 just / already / yet）
例句翻译
I've just finished cooking.我刚做完饭。
We have already tried oysters.我们已经吃过生蚝了。
四、关键时间标志词
ever 曾经 never 从未 before 以前 just 刚刚 already 已经 yet 还（否定/疑问） since + 过去时间 for + 一段时间 so far 到目前为止 in my life 这辈子
五、禁忌：不能搭配具体过去时间
❌ I've been there in 2019.
✅ I was there in 2019.（有明确过去年份，只用一般过去时）
六、have been to vs have gone to 极易混
结构含义人现在在哪例句
have been to + 地点 去过某地 ✅ 已回来 I've been to Milan.
我去过米兰，现在不在那。
have gone to + 地点 去了某地 ❌ 还在当地 She has gone to Hong Kong.
她去中国香港了（现在人在那）。
记忆方法：been = "去过"（已回来）/ gone = "去了"（还没回来）
就像中文"我去过北京"vs"他去了北京"，一个强调经历，一个强调人不在
七、I've been there 完整场景
场景1：
A: Do you know Milan?
B: Yes, I've been there.
🇨🇳 A：你了解米兰吗？ B：了解，我去过那儿。（现完：拥有经历，现在熟悉这座城市）

场景2：
A: Let's travel to Maldives this year.
B: I've been there, maybe we pick another place.
🇨🇳 A：我们今年去马尔代夫吧。 B：我去过了，换个地方吧。（现在结果：不想再去）
🚀 三种将来时结构完整拆解 重点突破
一、be going to 表将来
结构公式
am/is/are + going to + 动词原形 do
be 动词变化：I → am  |  he/she/it/单数 → is  |  you/we/they/复数 → are
核心用法：提前计划好的将来
动作是说话前已经规划、打算好要做的事
拆解：I'm going to buy new stationery.
I'm = I am（be 动词适配主语 I） |  going to = 将来结构标记  |  buy = 动词原形

句意：我打算去买新文具（买文具这个想法之前就想好的）
二、will 表将来
结构公式
will + 动词原形 do
will 无人称变化，所有人称都直接用 will（I will / She will / They will）
核心用法：临时决定 / 对未来预测
不是提前规划，是说话当下临时敲定的安排，或是对未来情况的预判
拆解：I will keep in touch with you.
will = 将来情态标记  |  keep = 动词原形

句意：我会和你保持联系（多是当下分别时临时做出的承诺/决定）
三、be doing（现在进行时表将来）
结构公式
am/is/are + 动词-ing 形式 doing
核心用法：已安排敲定好的近期将来
多用于位移类、会面类动词（meet / leave / come / go 等），日程已经确定、时间地点都安排妥当，近期就会发生
拆解：Rachel is meeting Diane next Friday.
is = be 适配单数主语 Rachel  |  meeting = 动词 -ing 形式  |  next Friday = 将来时间状语

句意：雷切尔下周五要和黛安碰面（会面计划已经敲定，时间、安排都定好了）
三者关键对比速记
结构适用场景Planning 程度例句
be going to 提前想好的计划、有迹象要发生的事 提前规划 I'm going to buy new stationery.
will 当场临时决定、纯未来预测 无提前规划 I will keep in touch with you.
be doing 日程完全敲定、时间固定的近期安排 安排最固化 Rachel is meeting Diane next Friday.
一句话区分：
昨天就想好要买 → be going to  |  此刻才决定的 → will  |  日历上已标好的 → be doing
🔀 条件句三种类型 必考
类型结构用法例句
零条件句 If + 一般现在时，一般现在时 客观事实 If you heat water, it boils.
第一条件句 If + 一般现在时，will + do 可能发生的未来 If it is sunny, we will go out.
第二条件句 If + 一般过去时，would + do 不太可能实现的假设 If I were rich, I would buy a house.
第三条件句 If + had done, would have done 过去未实现的遗憾 If I had known, I wouldn't have made mistakes.

🔄 双宾语 to / for 高频
公式：动词 + 人 + 物 = 动词 + 物 + to/for + 人
介词含义常见动词例句
to 方向/传递：东西从A到B lend, loan, give, pass, send, show, tell Pass me the salt.
= Pass the salt to me.
for 目的/服务：东西为B准备 buy, make, book, bring Buy me a coffee.
= Buy a coffee for me.
记忆口诀：传递给某人→to；为某人做→for
🎭 -ed / -ing 形容词系列 易错
动词-ed（人感受）-ing（事物属性）搭配介词
disappoint disappointed 失望的 disappointing 令人失望的 at sth / with sb / about sth
excite excited 兴奋的 exciting 令人兴奋的 about sth
surprise surprised 惊讶的 surprising 令人惊讶的 at sth / about sth
interest interested 感兴趣的 interesting 有趣的 in sth
tire tired 疲惫的 tiring 累人的 of sth 厌烦 / from sth 劳累

核心规律：-ed 描述人的感受 → "我感到……"
-ing 描述事物本身 → "这个东西令人……"

⚡ 情态动词 must / have to 家族 MC38
表达含义例句

🟣 have not / no 用法详解（not vs no）

核心规则：not 是副词，只能否定动词；no 是形容词，直接修饰名词。

错误正确说明
hasn't friendshas no friendsnot不能直接否定名词，用no+名词
hasn't friendshasn't got any friends (英式)haven't got + any + 名词
hasn't friendsdoesn't have any friends (美式)doesn't have + any + 名词
I haven't timeI have no time / I don't have timenot不能直接否定名词time
I haven't moneyI have no money / I don't have moneynot不能直接否定名词money
She hasn't friendsShe has no friends / She doesn't have any friends三单用doesn't have
There aren't problemsThere are no problems / There aren't any problemsthere be否定同样用no或any

📐 三种正确否定have的方式：

① no + 名词 → I have no friends.（最简洁）
② haven't got + any + 名词 → I haven't got any friends.（英式常用）
③ don't/doesn't have + any + 名词 → I don't have any friends.（美式常用）

❌ 绝对不能：

haven't / hasn't + 直接加名词（如 hasn't friends, haven't time）
→ 因为 not 是副词，不能像形容词一样直接修饰名词！

💡 not vs no 对比总结：

词性否定对象例句
not（副词）动词is not / do not / can not
no（形容词）名词no friends / no time / no money

记忆口诀：not配动词，no配名词 — not是副词管动词，no是形容词管名词。
