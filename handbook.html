🕐 时态体系总览 核心
⬅ 过去 已经发生的事
一般过去时
结构：did（过去式）
过去完成的动作
📌 I took part in a competition.
过去进行时
结构：was/were + doing
过去某一刻正在做
📌 I was minding my own business.

一般现在时
结构：do / does
习惯、爱好、事实
📌 I like jazz.
现在进行时
结构：am/is/are + doing
此刻正在做
📌 I'm living here.
现在完成时
结构：have/has + done
过去发生、影响现在
📌 I've been there.

be going to

🎯 现在完成时 have/has + done 完整解析 重点突破
一、核心公式
句型结构例句
肯定句 主语 + have/has + 过去分词 I have visited Milan.
否定句 have/has + not + 过去分词 I haven't been there.
疑问句 Have/Has + 主语 + 过去分词? Have you been to Hong Kong?
have vs has：三单主语（he/she/it/单数人名）→ has；其余（I/you/we/they/复数）→ have
两种形式：① have/has + 实义动词过去分词（普通动作）② have/has + been（be 动词的完成式）
二、核心本质：过去动作 → 留下现在结果/影响
不关心动作具体哪一天发生，只看重两点其一：
① 动作发生在过去，但对当下有影响
② 动作从过去持续到现在
时态例句侧重点
一般过去时 I went there in 2019. 带过去时间，只陈述当年去过，和现在无关
现在完成时 I've been there. 不带过去时间，重点：我现在拥有「去过那里」的经历
三、4大用法
用法1：曾经有过某种经历（最常用）
含义：人生到目前为止做过某事，常搭配 ever / never / before
拆解：I've been there.
I = 主语  |  have = 助动词（现完标志） |  been = be 的过去分词（be → was/were → been） |  there = 地点状语

直译：我到过那里 / 我去过那儿
现在影响：我现在熟悉这个地方，不用给我介绍；我有这段旅行经历
例句翻译
She's been to Australia.她去过澳大利亚。
Have you ever been to Hong Kong?你去过中国香港吗？
固定搭配：have been to + 地点 = 去过某地，现已回来
用法2：过去动作，结果留在当下（有直观影响）
例句翻译现在影响
I have lost my wallet.我钱包丢了。现在没钱，没法付款
We have eaten dinner.我们吃过晚饭了。现在不饿，不用点餐
用法3：动作从过去持续到现在（常搭配 for / since）
例句翻译时间起点
We have lived together for six years.我们同居六年了。2020年开始，持续到现在
I have worked in Shenzhen since March 2019.我2019年3月起就在深圳工作。至今还在
for vs since：for + 一段时间（for 3 years）/ since + 过去时间点（since 2019）
用法4：刚刚完成的动作（搭配 just / already / yet）
例句翻译
I've just finished cooking.我刚做完饭。
We have already tried oysters.我们已经吃过生蚝了。
四、关键时间标志词
ever 曾经 never 从未 before 以前 just 刚刚 already 已经 yet 还（否定/疑问） since + 过去时间 for + 一段时间 so far 到目前为止 in my life 这辈子
五、禁忌：不能搭配具体过去时间
❌ I've been there in 2019.
✅ I was there in 2019.（有明确过去年份，只用一般过去时）
六、have been to vs have gone to 极易混
结构含义人现在在哪例句
have been to + 地点 去过某地 ✅ 已回来 I've been to Milan.
我去过米兰，现在不在那。
have gone to + 地点 去了某地 ❌ 还在当地 She has gone to Hong Kong.
她去中国香港了（现在人在那）。
记忆方法：been = "去过"（已回来）/ gone = "去了"（还没回来）
就像中文"我去过北京"vs"他去了北京"，一个强调经历，一个强调人不在
七、I've been there 完整场景
场景1：
A: Do you know Milan?
B: Yes, I've been there.
🇨🇳 A：你了解米兰吗？ B：了解，我去过那儿。（现完：拥有经历，现在熟悉这座城市）

场景2：
A: Let's travel to Maldives this year.
B: I've been there, maybe we pick another place.
🇨🇳 A：我们今年去马尔代夫吧。 B：我去过了，换个地方吧。（现在结果：不想再去）
🚀 三种将来时结构完整拆解 重点突破
一、be going to 表将来
结构公式
am/is/are + going to + 动词原形 do
be 动词变化：I → am  |  he/she/it/单数 → is  |  you/we/they/复数 → are
核心用法：提前计划好的将来
动作是说话前已经规划、打算好要做的事
拆解：I'm going to buy new stationery.
I'm = I am（be 动词适配主语 I） |  going to = 将来结构标记  |  buy = 动词原形

句意：我打算去买新文具（买文具这个想法之前就想好的）
二、will 表将来
结构公式
will + 动词原形 do
will 无人称变化，所有人称都直接用 will（I will / She will / They will）
核心用法：临时决定 / 对未来预测
不是提前规划，是说话当下临时敲定的安排，或是对未来情况的预判
拆解：I will keep in touch with you.
will = 将来情态标记  |  keep = 动词原形

句意：我会和你保持联系（多是当下分别时临时做出的承诺/决定）
三、be doing（现在进行时表将来）
结构公式
am/is/are + 动词-ing 形式 doing
核心用法：已安排敲定好的近期将来
多用于位移类、会面类动词（meet / leave / come / go 等），日程已经确定、时间地点都安排妥当，近期就会发生
拆解：Rachel is meeting Diane next Friday.
is = be 适配单数主语 Rachel  |  meeting = 动词 -ing 形式  |  next Friday = 将来时间状语

句意：雷切尔下周五要和黛安碰面（会面计划已经敲定，时间、安排都定好了）
三者关键对比速记
结构适用场景Planning 程度例句
be going to 提前想好的计划、有迹象要发生的事 提前规划 I'm going to buy new stationery.
will 当场临时决定、纯未来预测 无提前规划 I will keep in touch with you.
be doing 日程完全敲定、时间固定的近期安排 安排最固化 Rachel is meeting Diane next Friday.
一句话区分：
昨天就想好要买 → be going to  |  此刻才决定的 → will  |  日历上已标好的 → be doing
⚡ be going to vs be doing（表将来）超详细对比 新
一、核心定义
be going to — 场景 1：有明显迹象，马上发生（客观预判）
肉眼能看到征兆，立刻会出现
Look! The boy is going to fall off the tree.
看！男孩马上要从树上摔下来。
Look at the dark clouds. It's going to rain.
看乌云，马上要下雨。
be going to — 场景 2：早已做好计划、打算（主观计划）
提前想好、有准备的安排
I am going to visit the flea market this Sunday.
这周日我打算去逛跳蚤市场（提前想好的计划）。
be doing（现在进行时表将来）
仅限位移类动词：go / come / leave / arrive / fly / travel / move
含义：已经敲定、定好时间的出行安排，日程固定，几乎不会变动
I'm leaving for Sydney next year.
我明年动身去悉尼（机票/行程全部定好）。
She's coming to see me tomorrow.
她明天过来看我（双方约好）。
二、3 大关键区别
区别 1：能用的动词范围
✅ be going to：所有动词
He is going to cook dinner. 他准备做饭。
He is going to fall. 他快要摔倒。
⚠️ be doing：仅出行/位移动词
❌ I'm cooking dinner tomorrow.
（cook 不是位移词，不能用）
✅ I'm going to cook dinner tomorrow.
区别 2：计划确定性
🔒 be doing：100% 敲定
行程完全敲定、有车票/约定，确定性最强
We're flying to Mexico tonight.
机票买好，行程固定
💭 be going to：只是打算
心里打算，不一定落实，可更改
I'm going to fly to Mexico someday.
只是想法，没订票
区别 3：预判即将发生
⛔ 只能用 be going to，绝对不能用 be doing
✅ Look out! The wall is going to collapse.
小心！墙马上要塌了。
❌ Look out! The wall is collapsing.
→ "墙正在塌"，不是预判将来
三、同一句话对比，直观看懂
表达含义关键区别
I'm going to visit my grandma. 我打算去看望奶奶 心里计划，还没定具体时间
I'm visiting my grandma this Saturday. 这周六我要去看望奶奶 约好时间，行程确定
四、will / be going to / be doing 三者极简总结
will do
临时突然决定
单纯猜测
无提前计划
This bag is heavy.
→ I will help you.
be going to do
有征兆马上发生
心里打算
适用所有动词
Look at clouds!
→ It's going to rain.
be doing
固定出行日程
仅位移动词
行程确定
I'm flying to Sydney.
机票已买好
五、超直白 2 条铁律（做题直接套）
铁律 1：动词限制
be doing 表将来
只能用：go / come / leave / arrive / fly / travel / move
❌ visit、cook、buy、fall、study… 一律不能用
be going to do
所有动词都能用，无任何限制
✅ 出行、做饭、逛街、摔倒、下雨，全部可以套
铁律 2：计划的确定程度
be doing（位移动词）be going to do
确定程度 100% 敲定 已买票、约好时间、板上钉钉 只是打算 心里想法，没落实票据/约定
例句 I'm leaving for Sydney next year.
机票租房全部搞定		 I'm going to leave for Sydney someday.
只是想法，没订票
六、两种绝对不能互换的场景
场景 1：预判马上发生（危险/天气）→ 只能用 be going to
✅ Look! He's going to fall off the tree.
看！他马上要从树上摔下来。
❌ Look! He's falling off the tree.
→ "他正在往下摔"（已经摔了，不是预判将来）
✅ Look at clouds! It's going to rain.
看乌云，马上要下雨。
场景 2：非出行类动作（逛街/做饭/学习）→ 只能 be going to
❌ I'm cooking dinner tonight.（cook 不是位移词，语法错误）
✅ I'm going to cook dinner tonight.
❌ I'm buying a new pen tomorrow.（buy 不是位移词）
✅ I'm going to buy a new pen tomorrow.
📝 做题口诀
眼见征兆马上发生 → 只用 be going to
出门行程定好日期 → 位移动词用 be doing
临时随口做个决定 → 直接选用 will do
极简做题口诀：
① 动词不是 go/come/leave/fly 这类出行词 → 只用 be going to
② 是出行动词，但只是心里想想、没订票 → be going to
③ 是出行动词，车票/约定全部定死、不改期 → be doing
④ 看见眼前征兆，马上要出事 → 只能 be going to
🔀 if 引导四大条件句 完整超详细讲解 核心
前置基础概念
if 条件句永远分为两部分：
• if 从句：提出假设的条件（放前/放后都可以）
• 主句：满足条件后产生的结果
四大类核心区分标准：假设是否真实、发生的时间、实现可能性
一、零条件句 Zero Conditional（100% 必然发生，客观真理）
1. 标准时态公式
If + 一般现在时，主句 + 一般现在时
2. 使用场景
描述永恒不变的客观规律、科学常识、固定规则、机器固有功能。无时间限制，过去/现在/未来只要满足条件，结果一定会发生。翻译常用「只要……就……」
3. 例句拆解
If you heat water to 100℃, it boils.
只要把水加热到100摄氏度，水就会沸腾。（科学定理，永远成立）
If you touch fire, you get burned.
只要触碰火焰，你就会被烫伤。
If you don't water plants, they die.
不给植物浇水，它们就会枯死。
4. 关键特征 & 易错点
• 主句绝对不能出现 will/can/may/might/be going to 这类表将来的词
• 只讲永久规则，不针对某一次、某一天的单独情况
❌ 错误：If you heat water, it will boil.
✅ 标准零条件句：If you heat water, it boils.
二、第一条件句 First Conditional（将来真实假设，高概率发生）
1. 标准时态公式
If + 一般现在时，主句 + will/can/may/might/be going to + 动词原形
2. 使用场景
针对未来某一次特定场景做出假设，条件有很大概率实现，用于计划、警告、承诺、可能性推测。
3. 主句换词含义（不同情态动词语气差异）
will：确定会做
can：能够、可以（具备条件就能做）
may/might：也许、可能（不确定，概率偏低）
be going to：提前计划好将要做
👉 主句换用 can/may/might/be going to 依然属于第一条件句，只是语气不同，不属于零条件句
4. 分层例句
✅ will（确定）：If it is sunny this Sunday, I will visit the flea market.
如果这周日天晴，我就去逛跳蚤市场。（本周日单次计划）
✅ can（能够）：If I finish homework early, I can watch TV.
如果我早点写完作业，我就能看电视。
✅ might（也许）：If it rains tomorrow, we might stay at home.
如果明天下雨，我们或许会待在家里。
✅ be going to（计划）：If I have money, I am going to buy a new bag.
如果我有钱，我打算买个新包。
5. 硬性语法规则
if 从句表将来时，永远不能用 will，只能用一般现在时：
❌ If I will have time, I go shopping.
✅ If I have time, I will go shopping.
6. 和零条件句核心区别
零条件句：永久通用  |  第一条件句：只针对未来单次事件
零：If it rains, the ground gets wet.（只要下雨地面就湿，永久规律）
第一：If it rains this Saturday, I can stay home.（仅限本周六这一次）
三、第二条件句 Second Conditional（现在/将来虚拟，几乎不可能）
1. 标准时态公式
If + 一般过去时，主句 + would/could/might + 动词原形
⚠️ 特殊规则：所有人称 be 动词统一用 were（正式语法）
If I were you / If he were rich
2. 两大使用场景（全部是空想，和现实相反）
场景 1：与【当下现在】的真实事实相反
现实是目前不具备这个条件，只是幻想当下改变现状。
真实：I don't have much money now. 我现在没钱。
虚拟：If I had much money, I would travel abroad.
如果我现在有钱，我就出国旅行。（暗含现实：我没钱，做不到）
真实：I am not tall. 我个子不高。
虚拟：If I were taller, I could play basketball well.
我要是高点，篮球就能打得很好。
场景 2：【未来极低概率】的幻想
中彩票、超难实现的愿望
If I won the lottery tomorrow, I would quit my job.
要是我明天中了彩票，我就辞职。（中奖概率极低，纯幻想）
3. 重要易错提醒
if 从句里的过去式不代表过去发生，只是虚拟语气的语法标记，专门表示「现在和现实相反」
If I had money ≠ 我过去有钱，意思是：我现在要是有钱（现在没钱）
四、第三条件句 Third Conditional（过去虚拟，无法挽回的遗憾）
1. 标准时态公式
If + had + 过去分词（had done），主句 + would/could/might + have + 过去分词（would have done）
2. 使用场景
针对已经结束、无法更改的过去进行假设。现实中过去已经发生了某件事，现在幻想当初换一种做法，会有不一样的结果，自带后悔、惋惜情绪。
3. 例句对照（真实过去 vs 虚拟假设）
真实：I got up late yesterday, so I missed the bus.
我昨天起晚了，错过了公交车（事情已经发生，无法重来）

虚拟：If I had got up early yesterday, I wouldn't have missed the bus.
我昨天要是早起，就不会错过公交车了。
真实：She didn't study hard, so she failed the exam last week.
虚拟：If she had studied hard, she could have passed the exam.
她当初要是努力学习，上周本来能及格。
4. 时间区分核心
第二条件句 → 聚焦现在、未来，现状能看见，只是空想改变当下
第三条件句 → 聚焦昨天、上周、从前，事情已成定局，无法补救
五、第二 vs 第三条件句直观对比（同一件事，不同时间）
第二条件句（现在没空）
If I had free time now, I would help you.
我现在要是有空，我就能帮你。
第三条件句（昨天没空）
If I had had free time yesterday, I would have helped you.
我昨天要是有空，当时就能帮你了。
六、四大条件句汇总表格
类型从句时态主句时态可能性适用场景
零条件句 一般现在时 一般现在时 100% 一定发生 客观真理、永久规则
第一条件句 一般现在时 will/can/may/be going to do 大概率发生 未来单次真实计划、警告
第二条件句 一般过去时 did would/could/might do 几乎不可能 当下现实相反、未来空想
第三条件句 had done 过去完成时 would/could/might have done 完全不可能（过去已定） 对过去的后悔、假设
📝 万能做题区分口诀
① 永久道理，主句无将来词 → 零条件句
② 未来单次计划，主句带 will/can/may → 第一条件句
③ 现在现状相反，从句过去式，主句 would + 原形 → 第二条件句
④ 过去已成定局，从句 had done，主句 would have done → 第三条件句
核心区分维度：
• 时间：现在/未来 vs 过去已成定局
• 真实性：100%真实 vs 大概率 vs 几乎不可能 vs 完全不可能
• 从句时态：现在时 → 零/第一  |  过去时（did）→ 第二  |  过去完成时（had done）→ 第三
📊 频率副词排序 MC36
never 0% → rarely → seldom → occasionally → sometimes → often → usually → always 100%
位置规则：实义动词前，be动词后
✅ We seldom fasten seat belts.（实义动词前）
✅ I am usually busy.（be动词后）
⚡ sometimes 位置灵活，可放句首/句中/句尾
🔄 双宾语 to / for 高频
公式：动词 + 人 + 物 = 动词 + 物 + to/for + 人
介词含义常见动词例句
to 方向/传递：东西从A到B lend, loan, give, pass, send, show, tell Pass me the salt.
= Pass the salt to me.
for 目的/服务：东西为B准备 buy, make, book, bring Buy me a coffee.
= Buy a coffee for me.

📋 so far vs by now 完整区分 新
一、so far
核心意思：到目前为止；迄今为止（从过去一直延续到说话此刻）
字面：到现在这个节点为止，全程累计的情况
1. 固定搭配时态（必考）
绝大多数搭配现在完成时 have/has done
2. 两层场景
场景用法例句
累计数量/进度 So far + 现在完成时，强调总共、累计 So far, I have learned 3000 English words.

目前情况评价 So far + 一般现在时，描述当下状态 So far, everything is good.
到目前为止，一切顺利。
3. 位置灵活
句首或句尾都可以：
So far, no one has come. = No one has come so far.
二、by now
核心意思：到现在这个时候；按理此刻应该……
自带两层隐含语气：
① 说话人心里有预期，觉得这件事现在本该发生
② 现实往往和预期相反（暗含失望、疑惑）
1. 时态搭配
语气结构场景
肯定/预期should have done虚拟，本该做却没做（暗含失望）
客观陈述现在完成时 have done说话当下的客观结果
口语一般现在时随意的日常表达
2. 例句（突出"预期反差"）

按说到这个点他们早就该到了。（现实：人还没来，失望）

五、同类型时间表达 全面对比
表达语体风格时态隐含预期例句

up to now 书面通用 现在完成时 🚫 无，纯累计 Up to now, we haven't received any reply.

📢 祈使句详解 语法
类型结构例句
肯定祈使句动词原形开头（无主语）Sit down, please. / Open the door.
否定祈使句Don't + 动词原形Don't be late. / Don't worry.
Let's 祈使句Let's + 动词原形 = 一起做某事Let's go! / Let's take a break.
Be + 形容词Be + adj. = 保持…状态Be careful! / Be quiet!
Please 礼貌Please + 祈使句 / 祈使句 + pleasePlease sit down. / Come in, please.
No + doing（禁止）No + 动名词 = 禁止做某事No smoking. / No parking.
祈使句核心：省略主语 you，直接用动词原形开头。否定加 Don't，礼貌加 Please。
Let's = Let us 缩写，表"让我们一起…"，反意疑问用 shall we？

🟣 have not / no 用法详解（not vs no）

核心规则：not 是副词，只能否定动词；no 是形容词，直接修饰名词。

错误正确说明

📐 三种正确否定have的方式：

① no + 名词 → I have no friends.（最简洁）
② haven't got + any + 名词 → I haven't got any friends.（英式常用）
③ don't/doesn't have + any + 名词 → I don't have any friends.（美式常用）

❌ 绝对不能：

haven't / hasn't + 直接加名词（如 hasn't friends, haven't time）
→ 因为 not 是副词，不能像形容词一样直接修饰名词！

no（形容词）名词no friends / no time / no money
